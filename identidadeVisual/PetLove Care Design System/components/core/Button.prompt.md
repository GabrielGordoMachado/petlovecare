Brand button for PetLove Care — coral primary action, teal secondary, plus danger / outline / link variants; use for any tap target.

```jsx
<Button variant="primary" size="lg">AGENDE AGORA</Button>
<Button variant="secondary">Entrar</Button>
<Button variant="danger" size="sm">Recusar</Button>
<Button variant="link">Esqueceu a senha?</Button>
```

- `variant`: `primary` (coral, default) · `secondary` (teal) · `danger` (red) · `outline` (coral border) · `link` (teal text link).
- `size`: `sm` · `md` · `lg`. `fullWidth` stretches to container.
- Display font (Poppins bold), uppercase brand CTAs read well at `lg`.
